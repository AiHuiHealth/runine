装机必备影音播放器 PotPlayer v1.7.21419 去广告绿色版

· 2021-02-10 · # 媒体工具 # 影音工具 # 视频播放 # 装机必备 # WIN

PotPlayer 是由 KMPlayer 的原作者姜勇希在跳槽到 Daum 公司后使用 C++ 完全重写的播放器。它基本上继承了 KMPlayer 方便的滤镜以及外挂式的管理系统。同时简化了内部的解码器体系。

软件介绍

曾几何时 KMPlayer 是无数视频、电影爱好者的首选免费全能视频播放器，在原作者姜勇囍离开并进入新公司 Daum 之后，他们推出了新一代的作品 PotPlayer 则拥有更先进和优异的特性。PotPlayer 几乎继承了 KMPlayer 所有的优点，拥有异常强大的内置音视频解码器，可以支持几乎全部音乐、视频文件格式的播放。采用 VC++ 进行了全面重构，并且原生支持64位操作系统，使得 PotPlayer 在 Windows 10 等新系统上，无论性能、兼容性和稳定性上的表现均比 KMP 要好不少，被无数人誉为当今 Win 平台上最优秀的免费本地影音全能格式播放器……

由于 PotPlayer 内置了非常全面且兼容性良好的视频音频解码器，因此用户无需进行任何手动配置，即可以直接播放几乎目前网络上所有主流的视频音频格式文件，非常方便。PotPlayer 的配置选项非常丰富，各种功能也非常强大，对字幕文件的支持也非常好。在开启了 PotPlayer 的硬件解码支持之后，你一定会惊叹于它的内存占用率和播放效果！

软件特性

◍ 支持多种加速引擎 DXVA, CUDA, QuickSync 等等，以最少的资源制造最华丽的表现。
◍ 支持文字字幕SMI，SRT，Vobsub(DVD)，蓝光SUP字幕等多种字幕资源。
◍ 支持 ASS/SSA 动画以及SMI ruby tag。
◍ 支持多种编解码器，无需安装各种全能编解码器
◍ 支持OpenCodec，用户也可以自定义添加编解码器。
◍ 多个文件一次性读取，流畅播放。
◍ 双声卡时，可自定义输出。
◍ 各时间段截图方便快速切换到不同场景。
◍ 提供特定场景添加并管理书签、章节功能。
◍ 支持Direct3D9 Ex Flip Mode, Overlay 功能。
◍ 支持其它硬件设备播放（DVD、 TV、 HDTV）。
