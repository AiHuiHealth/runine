使用油猴脚本和 IDM 或迅雷下载百度网盘限制大文件

· 2020-07-11 · # 文章教程 # 浏览器

我们都知道百度网盘是目前国内最大的网盘，使用人数也是最多，相比其它网盘来说，百度网盘还是比较稳定，一般不会出现什么乱子。

最近几年百度网盘为了盈利，现在下载百度网盘资源文件一般大些的都会提示要求安装百度网盘，这对于很多用户来说非常不方便。而且百度网盘即使安装了想要提高下载速度也是需要购买 VIP 会员加速才可以。

虽然这两年也有一些作者开发了第三方的百度网盘下载工具，但是有些用户却可能会遭遇封禁帐户的问题。今天给大家推荐的是使用 Tampermonkey 油猴脚本配置 IDM 或其它下载工具来下载百度网盘里的资源。

Tampermonkey 油猴脚本是一款免费的浏览器扩展程序，它本身也可以添加很多不同的脚本，是浏览器上的一个脚本管理集成工具。借助所编写的脚本它可以做很多事情，比如：下载各大视频网站的VIP 视频、去除各大视频广告，百度搜索广告、免费收听网易云320k 高音质音乐、高速下载百度云、去除各种验证码等等。

Internet Download Manager 下载安装

什么是 IDM? Internet Download Manager，简称 IDM，是国外的一款优秀下载工具，该软件同时是一款共享软件。当然这里给大家提供的是 IDM 特别版，直接安装即可。

点击下载 IDM 中文版：https://git.io/JJYY8

Tampermonkey 油猴脚本安装

官网：https://tampermonkey.net/

这里以 Chrome 为例。其他浏览器比较简单（直接去应用商店里搜索”Tampermonkey”或“油猴”）

下载 Tampermonkey 油猴脚本安装包。

在 Chrome 中，在地址栏输入：chrome://extensions/ 回车，打开应用管理器，可以看到之前安装的其他应用。

拖放刚刚的 Tampermonkey 安装包到界面里，松开鼠标，会弹出安装提示，点击确定安装就可以了
