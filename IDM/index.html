Internet Download Manager 是一款曾获国外著名IT网站最高5星评价的下载软件，IDM即使是在国内，也深受很多资深电脑用户喜欢。特别是配合脚本利用IDM下载百度云盘文件，速度也非常惊人 。

软件介绍

IDM是一个方便的下载工具，该软件提升你的下载速度最多达5倍，安排下载时程，或续传一半的软件。Internet Download Manager 的续传功能可以恢复因为断线、网络问题、计算机宕机甚至无预警的停电导致下传到一半的软件。此程序具有动态档案分割、多重下载点技术，而且它会重复使用现有的联机，而不需再重新联机登入一遍。聪明的 in-speed 技术会动态地将所有设定应用到某种联机类型，以充分利用下载速度。

Internet Download Manager 支持下载队列、防火墙、代理服务器和映射服务器、重新导向、cookies、需要验证的目录，以及各种不同的服务器平台。此程序紧密地与 Internet Explorer 和NetscapeCommunicator 结合，自动地处理你的下载需求。此程序还具有下载逻辑最佳化功能、检查病毒，以及多种偏好设定。

Internet Download Manager 下载 HTTP 链接十分快，而且很稳定，凭借的独特的下载方式不会出现网速时快时慢的情况。虽然它不支持P2P协议，但是在BT和电驴逐渐不再流行的情况下，IDM以其“网盘利器”的美名迅速蹿红。支持多款浏览器，包括IE，Safari，谷歌浏览器，火狐，MSN还有opera 。

同时IDM的文件名识别非常准确，当服务器端的文件名和下载文件名不同，Internet Download Manager 也会提示你改正。但是可以通过搭配下载的方式来使用IDM（比如用迅雷离线下载好的资源，然后结合 HTTP 协议的特性再用 Internet Download Manager 下载）可以达到事半功倍的效果。

软件特性

  1. 支持多种类型的代理服务器。例如，IDM与Microsoft ISA和FTP代理服务器一起工作。
  2. 支持主要的身份验证协议：基本、协商、NTLM和Keberos。因此，IDM可以使用登录名和密码访问许多Internet和代理服务器。
  3. 下载所有功能。IDM可以添加链接到当前页面的所有下载。使用此功能可以轻松下载多个文件。
  4. 可定制的界面。您可以选择顺序，以及在主IDM窗口上显示哪些按钮和列。下载类别。Internet下载管理器可用于使用定义的下载类别自动组织下载。
  5. 快速更新功能。快速更新可以检查IDM的新版本并每周更新一次IDM。下载限制。逐步下载配额功能。此功能对于使用某种公平访问策略（或FAP）的连接很有用，如Direcway、Direct PC、Hughes等。
  6. 拖放。您可以简单地将链接拖放到IDM，并将完整的文件从Internet下载管理器中拖放。
  7. IDM是多语言的。IDM翻译成阿拉伯语、波斯尼亚语、保加利亚语、汉语、克罗地亚语、捷克语、丹麦语、荷兰语、法语、德语、匈牙利语、意大利语、日语、立陶宛语、马其顿语、波兰语、葡萄牙语、罗马尼亚语、俄语、斯洛文尼亚语、西班牙语、泰语。
