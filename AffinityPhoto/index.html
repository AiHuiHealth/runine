图像处理软件 Serif Affinity Photo v1.9.0 官方正式版

· 2021-02-05 · # Serif # 图形图像 # 图像处理 # 设计软件 # MAC # WIN

Serif Affinity Photo 是英国 Serif 公司历时5年开发的一款强大的图像处理软件，功能上类似于 Adobe Photoshop，最初只有 Mac版，软件推出后获得用户的一致好评，后来推出 Windows 版。

软件介绍

Serif Affinity Photo 是一款极具专业图像处理编辑软件。软件采用了比Adobe更先进的算法，为用户提供了非常精密的工具可以让用户轻松的编辑、修饰你的照片。而且软件还有非常专业的校正和调整，用户可以利用软件里面的色阶、曲线、白平衡、HSL等十几种功能进行无损调整修改图片，而且再调整过程中还能实时进行预览哦。软件还有非常齐全的修饰工具，可以对图片进行加深、仿制、修补、斑点修复等等功能。此外，软件还有专用的RAW工作空间、PSD导入/导出、批处理、全液化、自然的笔法、照片拼接等等功能。

Serif Affinity Photo 已成为全球摄影和创意专业人士的首选工具，他们喜欢它的速度、功能和精确度。它与最新的强大计算机技术携手合作，是唯一一款集成在 macOS、Windows 和 iOS 上的全负载照片编辑器。
