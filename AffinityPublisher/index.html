设计出版软件 Serif Affinity Publisher v.1.9.0 官方正式版

· 2021-02-05 · # Serif # 图形图像 # 图像处理 # 设计软件 # MAC # WIN

Serif Affinity Publisher 是由英国 Serif 公司推出的一款专业级专业出版软件，功能上类似于 Adobe InDesign，最初只有 Mac版，软件推出后获得用户的一致好评，后来推出 Windows 版。

软件介绍

Serif Affinity Publisher 是一款专业的出版应用软件，可以用于杂志、书籍、小册子、海报、报表等内容的排版工作，支持图文混排、首字下沉、插入表格、从母版创建页面等常用的排版功能。它是继Photo和Designer之后的有一款逆天排版神器，可以进行专业的排版工作，支持图文混排、首字下沉、插入表格、从母版创建页面等常用的排版功能。该软件旨在帮助专业设计人员在每一版面、页面、杂志、书籍和数字出版物中实现最佳的效果，展现令人惊艳的排版和绚丽的色彩。

Serif Affinity Publisher 是新一代的专业出版软件。它针对 Windows 和 Mac 上的最新技术进行了优化，并被苹果评选为“年度 Mac 应用”。借助这款无比流畅直观的应用，无论是书籍、杂志和营销材料，还是社交媒体模板、网站模型等，您都可以轻松地将图像、图形和文本整合为一体，打造出版所需的精致布局。
