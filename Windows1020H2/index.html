装机必备系统镜像 Windows 10 20H2 纯净多合一精简版

· 2021-02-17 · # 系统映像 # 系统相关 # 系统镜像 # WIN

Windows 10 20H2 纯净多合一精简版ISO镜像，由歪果不求仁精简封装制作，纯净无任何三方软件及个人OEM信息。此精简优化版合集，含专业版、企业版、教育版及专业工作站版，能正常更新补丁。




系统特性

◍ 更新补丁功能未测试，发布作品前还没有收到后面的推送的补丁。
◍ 纯净 无任何第三方软件。
◍ 追求稳定、没有过度优化，可以正常办公。
◍ 打印 蓝牙 指纹 共享 微软账户 计算器 屏幕投影 平板模式 都能正常使用。(需要用到 指纹 面部识别 微软账户 位置服务的 请用自建账户版，Admin内置管理员版不支持)。
◍ 避免以后升级到更高版本的系统，所有网络已经设置为按流量计费，检查更新后 下载补丁需经过你同意，但不影响系统给设备打驱动功能。
◍ 自建账户版保留任务栏搜索需要的自行设置显示即可，Admin内置管理员版未保留。
◍ 默认关闭7G预留空间。
◍ 图片文件默认不使用“画图”打开。
◍ 恢复要 “使用此计算机必须输入用户密码”的勾选框，原版给隐藏了。
◍ 自建账户版保留 windows hello( 面部识别) 指纹，Admin内置管理员版未保留。
◍ 关闭自动播放
◍ IE主页默认设置为百度，原版是先打开微软再跳转百度。
◍ 由于集成了补丁，DISM++ 不可以正常清理被取代的SXS组件。
◍ Windows Store应用商店可以一键恢复或者卸载，并提供了应用商店安装包。
◍ 提供 Microsoft Edge浏览器程序安装包，网上也可以下载到。
◍ 提供 OneDrive微软云盘程序安装包，网上也可以下载到。
◍ Admin内置管理员版 已经精简地图控制、商店照片应用 查看文件信息会闪退，自建账户不影响。
◍ 右键添加 管理员取得所有权
◍ 右键添加 复制文件路径
◍ 右键添加 复制文件夹路径
◍ 如需取消右键 管理员取得所有权 、复制文件路径 文件夹路径 请用CMD 运行以下命令即可删除：
◍ 集成 .NET Framework 3.5

版本说明

Admin内置管理员版

自建账户版
映像名称没有 (Admin) 标识为自建账户版，自建账户版顾名思义安装时需要自己创建一个账户。
