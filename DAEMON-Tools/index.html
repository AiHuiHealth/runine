DAEMON Tools 是一款功能专业的映像制作软件，虚拟光驱工具，支持创建/编辑/刻录/提取，数据映像、音频映像、加密光盘、虚拟硬盘等功能，支持创建和加载读取各种虚拟光驱的映像文件类型格式。

软件介绍

DAEMON Tools Ultra 是一款功能专业的虚拟光驱软件，映像制作软件。这款映像处理工具，加载大型镜像文件非常快！支持创建/编辑/刻录/提取，数据映像、音频映像、加密光盘、虚拟硬盘、虚拟光盘、内存磁盘、VHD、USB引导盘、ISCSI介质、虚拟设备装载等功能，支持创建和加载读取各种虚拟光驱的映像文件类型格式。而且软件还能同时运行多个向导，可以大量节省用户的时间。此外，软件还有加密文件、创建可启动的USB、保护USB、添加可写虚拟设备等等各种功能。

软件特性

◍ 可以在软件中创建一个数据映像，可以将ISO的映像添加都软件上
◍ 创建音频图像，可以将的加载的音频文件数据和数据图像显示在电脑上
◍ 支持烧录功能，DAEMON Tools 提供了一个刻录的功能，可以将ISO输出到光盘
◍ 支持光盘数据复制功能，可以将电脑上插入的光盘数据复制到PC上
◍ 加密文件，当您在编辑ISO或者是转换一个数据映像的时候，可以添加密码
◍ 添加可写的虚拟设备，创建硬盘虚拟分区，可以让您在编辑数据映像的时候更加方便
◍ 使用iSCSI目标面板中你可以使用虚拟光驱超不仅设备在你的电脑也是在iSCSI服务器共享任何远程设备。
