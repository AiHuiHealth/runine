装机必备 微软常用运行库合集 v2021.01.08 x32/64

· 2021-01-08 · # 系统辅助 # 系统增强 # 系统相关 # 装机必备 # 系统工具 # WIN

微软常用运行库合集32/64位这些运行库都是采用Microsoft Visual Studio 20XX编写的软件必须使用的公用DLL运行库，相当于程序的字典文件；只需要一键全部安装，堪称装机必备神器！

软件介绍

运行库就是支持大部分程序运行的基础，由于很多常用软件都是采用 Microsoft Visual Studio 编写的，所以这类软件的运行需要依赖微软Visual C++运行库，比如像 QQ、迅雷、Adobe 软件等等，如果没有安装VC++运行库或者安装的版本不完整，就可能会导致这些软件启动时报错，提示缺少库文件。然而，很多朋友却不知道到底要安装哪些运行库版本。微软常用运行库合集就是为了解决安装运行库的难题。此微软系统运行库集合包整合了目前最新的、最常用的VC运行库版本，一键全部安装，方便操作，堪称装机必备神器！

对于新装机的朋友，事先将一些系统必备的常用运行库安装齐全，可以避免日后启动软件或运行游戏时出现各种错误，特别是帮 MM、家长们装机更应该去做这一点。不过逐一去下载安装这些库实在太麻烦了！这时微软常用运行库合集就是你的救星啦！则可帮你一次全部自动安装……

此工具不仅可以让用户不再遇到因缺少运行库导致的程序错误或游戏启动失败等各种烦人问题。安装系统之后也不再需要再一个个单独的去找运行库，省时省力！而且可以离线安装，绝对是装机必备的好工具。

如果提示缺少DLL组件的可以尝试使用：全能运行库修复工具 DirectX Repair 进行修复，大多数的缺少DLL组件都可以使用此软件进行修复。

安装参数

系统封装爱好者，可利用静默参数调用安装!

微软常用运行库合集版
/SILENT 显示窗口直接安装
/VERYSILENT 静默安装

VisualCppRedist版
/ai 自动安装所有，不显示任何输出
/y 自动安装所有，有安装界面进度显示
/aiM 自选安装所有，安装每个版本有提取对话框，并确认询问
/aiU 自动卸载所有，卸载每个版本有提取对话框，并确认询问
/aiA 自动安装所有，但隐藏ARP项
/aiR 自动卸载所有
/ai5 自动安装Microsoft Visual C ++2005
/ai8 自动安装Microsoft Visual C ++2008
/aiX 自动安装Microsoft Visual C ++2010
/ai2 自动安装Microsoft Visual C ++2012
/ai3 自动安装Microsoft Visual C ++2013
/ai9 自动安装Microsoft Visual C ++2019
/aiT 自动安装Microsoft Visual Studio 2010
/aiE 自动安装额外的Microsoft Visual Basic/C++
/aiV 自动安装Microsoft C Runtime Library基础库
/aiH 系统“卸载程序”界面中“自动隐藏已经安装”项
/aiP 系统“卸载程序”界面中“手动隐藏或显示已经安装”项
/aiC 完全自动被动模式。安装所有，但UCRT KB3118401除外
/aiD 调试模式，创建VCpp_debug.log而不安装/卸载任何组件
/gm2 用于禁用所有其它开关的提取解压对话框的可选开关
