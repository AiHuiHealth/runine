开源播放器 ZY Player v2.7.1 全网VIP剧集/电影免费观看

· 2021-01-02 · # 媒体工具 # 视频播放 # 装机必备 # MAC # WIN

ZY Player 是一款基于electron开发的一款开源跨平台视频资源播放器，奇特点爬取全网视频资源聚合，一键搜索全网影视剧集无广告播放。并且资源丰富，如果你是一个影迷那推荐你使用。

软件介绍

ZY Player 资源播放器，免费无广告全网影视资源播放器，聚合大量影视网站搜索接口，奇特点爬取全网视频资源聚合，一键搜索全网影视剧集无广告播放，还支持第三方播放器播放，提供海报模式和列表模式浏览，可以自定义视频源和导出资源下载链接。是时候和“矮油疼”VIP说拜拜！

软件特性

影剧丰富
内置了最大资源网，OK资源网，速播资源站，麻花资源，最新资源网等20个视频资源网站的播放源。
高效浏览
浏览全网热门视频， 支持切换视频源；详细的电影分类，支持搜索电影名和演员名称。
智能播放
支持视频连播， 支持多种播放速度， 支持历史记录， 支持多种播放模式， 支持精简模式(划水模式)。
收藏分享
一键收藏分享喜爱的视频， 支持微信播放； 一键同步视频信息，追剧更简单。
资源下载
部分视频源支持下载 (最大资源网， OK资源网)
其他
多主题， 多语言， 自动更新

◍ 支持全平台：Windows, Mac, Linux

◍ 支持第三方播放器播放
◍ 显示豆瓣评分

常见问题

# ZY Player 资源加载不出来或无法显示
◒ 转到设置界面 >> 点击 “编辑源” / “编辑直播源” >> 点击 “重置” >> 重启软件。
◒ 转到设置界面 >> 点击 “软件重置” 恢复默认设置；但是会清空用户数据，如非必要不建议使用！

# ZY Player 视频无法播放
◒ 一般情况是资源网服务器不稳定, 或者资源网屏蔽了IP, 请尝试切换到其他视频源。
