专业磁盘性能测试工具 CrystalDiskMark v8.0.0 中文便携版

· 2020-12-17 · # 磁盘工具 # 硬件工具 # 硬件检测 # 系统工具 # WIN

CrystalDiskMark 是一款免费的专业磁盘性能测试工具。它是由日本人hiyohiyo开发的硬盘基准测试软件，用于检测机械硬盘及固态硬盘的读取和写入速度，混合随机读写、持续传输率、随机4K性能测试。

软件介绍

CrystalDiskMark 是一款测试电脑硬盘的工具，简单易于操作的界面让你随时可以测试你的存储设备。这款软件能够检测硬盘的随机读取/写入速度，以及连续读取/写入速度。测试随机512KB，2KB，4KB读取/写入速度等，用户可以以此来辨别磁盘的性能及质量的优劣程度。CrystalDiskMark体积小巧，用户界面直观简单。

软件特性

测试可持续和随机性能（读/写/混合）
高峰/稳定状况表现性能
第一栏 -> 当前测试的硬盘分区
第二栏 -> 当前硬盘的读写速度
第三栏 -> 硬盘4K文件多线程随机读写速度
第四栏 -> 硬盘的连续读写速度
第五栏 -> 硬盘4K小文件单线程随机读写速度
