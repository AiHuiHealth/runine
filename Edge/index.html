浏览器 Microsoft Edge v86.0.622.63 精简增强便携版

· 2020-11-10 · # 网络工具 # 装机必备 # WIN # 浏览器

Microsoft Edge 是由微软开发的一款基于Chromium内核运行在Windows 10系统上的新型浏览器。Edge浏览器的交互界面更加简洁，还将兼容现有Chrome与Firefox两大浏览器的扩展程序。

软件介绍

Microsoft Edge 是随着Win 10系统开发而推出的的一软微软浏览器，而在Win 10 之前的系统中使用的是IE浏览器。Edge浏览器内核是使用微软全新开发的MSHTML渲染引擎，与IE浏览器完全不同，Edge浏览器要比IE11浏览器速度快两倍多，Edge浏览器比IE更流畅、外观UI相对于IE还是比较舒服的。依赖于Win10系统。可以在网页上撰写或输入注释，并与他人分享。IE浏览器可以向下兼容，导致上网速度比较慢，且由于支持旧技术，安全性也难以经受考验，CPU使用率和耗电量更低。

软件特性

兼容性强
Microsoft Edge作为微软新一代的原生浏览器，在保持IE原有的强大的浏览器主功能外，还完美补充了扩展等现代浏览器功能。重构后的 Microsoft Edge 浏览器为您带来出色的兼容性及性能，守护您的安全及隐私，同时新功能旨在为您带来更好的网络体验。新版 Microsoft Edge 浏览器现可在所有受支持的 Windows、macOS、iOS 和 Android 版本上使用。

保护隐私
借助防追踪功能，Microsoft Edge 浏览器让您能够更好地控制数据，在您浏览时，提高追踪器被拦截的透明度。在新版 Microsoft Edge 浏览器中使用 InPrivate 模式，您的浏览历史和搜索记录不会归于是您所为或保存到您的设备，从而使您可以更好地管控数据。

安全浏览
Microsoft Defender SmartScreen 可以自动保护您在线时免受安全问题、网络钓鱼和恶意软件的侵害。NSS Labs 最近进行的两项独立研究发现，Microsoft Edge 对恶意软件和网络钓鱼的防护能力最强。当您的凭据泄漏时，密码监视器会通知您，以便您可以更新凭据并确保个人信息安全。

人性化体验
新版 Microsoft Edge 浏览器有您所期待的速度和性能，同时与您喜爱的网站和扩展有着出色的兼容性。从其他浏览器导入收藏夹、密码等内容，使所需的一切都存储在一个地方，从而让您以适合自己的方式进行浏览。

自定义扩展
新版 Microsoft Edge 浏览器现可兼容您最喜爱的扩展程序，因此可以轻松个性化您的浏览体验。
