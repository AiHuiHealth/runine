前言

Windows Defender 是 Windows10 自带一款安全防护软件，对这款系统自带的安全防护软件网上有不少诟病，收录最多的关键词都是怎么关闭、怎么移除！对于喜欢系统封装优化的小伙伴来说，那是必须移除的必选项！先不说Windows Defender有什么独特的功能，非常影响性能不说，很多时候我们下载的软件或补丁直接会被删除或者无法运行，对于小白来说都不知道从哪里恢复加入白名单，所以非常不符合中国国情，所以歪果每次装完系统都会毫不客气的卸载掉Windows Defender，选择安装第三方安全防护软件。以下歪果给大家提供了几种可选的应对方法！

使用手动关闭

暂时关闭 Windows 安全中心
只是临时关闭关闭 Windows 安全中心，电脑重启后会重置开启！

设置(Win+i) -> 更新和安全 -> Windows 安全中心 -> 打开 Windows 安全中心 -> 病毒和威胁防护 -> 管理设置 -> 依次关闭 “实时保护” + “云提供的保护” + “自动提交样本” + “篡改防护” 即可！

永久关闭 Windows 安全中心
永久性移除 Windows 安全中心，但是可能系统更新时会反弹！

1. 设置(Win+i) -> 更新和安全 -> Windows 安全中心 -> 打开 Windows 安全中心 -> 病毒和威胁防护 -> 管理设置 -> 依次关闭 “实时保护” + “云提供的保护” + “自动提交样本” + “篡改防护” 。
2. 右键开始菜单选择运行(Win+R) -> 输入命令 “gpedit.msc” 打开本地组策略 -> 管理模板 -> Windows 组件 -> Windows Defender 防病毒程序 -> 关闭 Windows Defender 防病毒程序 -> 已启用 -> 应用/确定即可。
