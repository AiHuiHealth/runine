装机必备运行库 Microsoft .NET Runtime v5.0.3 官方正式版

· 2021-01-15 · # 系统辅助 # 系统增强 # 系统相关 # 装机必备 # WIN

Microsoft .NET Framework 是Windows操作系统用于.NET Framework构建的应用程序所需的基础类型库组件。NET Framework是继Windows DNA之后的一个全新软件开发平台，支持多国语言的开发环境。

前言

目前，只需要电脑上有 3.x 和 4.x 版本，基本上就能运行所有.NET程序了。Windows 10 用户基本上无需再次安装.NET，Windows 7 用户建议安装4.5版本即可。如果安装时提示已有此版本，说明此电脑已经安装了该版本，无需额外安装。

Windows 10 用户无需额外安装 .NET3.5或4.8
可打开控制面板 -> 程序和功能 -> 启动关闭 Windows 功能 -> 在该界面找到.net版本开启即可！
如果您使用的系统为精简版本，可以下载以下歪果为你整理的官方离线安装包安装。

.NET Runtime -> 此版本仅包含运行控制台应用程序所需的组件。
.NET Desktop Runtime -> 此版本包含.NET Runtime，您无需单独安装。
ASP.NET Core Runtime -> 此版本用于运行Web/服务器构建类型程序。
.NET Framework -> 用于构建在Windows上运行的任何类型的应用程序。
